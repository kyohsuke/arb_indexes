# arb_indexes - get indexes from Model
===========

## Usage
put the Gemfile
```ruby
group :development do
  gem 'arb_indexes', :git => 'git://github.com/kyohsuke/arb_indexes.git'
end
```

kick it on rails console

```ruby
User.indexes
```
